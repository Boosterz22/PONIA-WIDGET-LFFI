Quick Start

Get started with Ponia in less than 60 seconds. Add the script tag to your HTML and trigger the widget with a button click.

Deposit Example

<!-- Add Ponia script -->
<script src="https://ponia.io/widget.js"></script>

<!-- Deposit button -->
<button onclick="openPonia({ targetChain: 'base', targetToken: 'USDC' })">
  Deposit with Ponia
</button>

Withdrawal Example

<!-- Withdrawal button -->
<button onclick="openPonia({ mode: 'withdraw', targetChain: 'solana', targetToken: 'USDC' })">
  Withdraw to Solana
</button>

Your users can now deposit from any chain OR withdraw to any chain with instant cross-chain transfers.

Installation

Via Script Tag (Recommended)

The easiest way to integrate Ponia is via our CDN-hosted script:

<script src="https://ponia.io/widget.js"></script>

Via NPM

For React, Next.js, or other modern frameworks:

npm install @ponia/widget

// Import in your component
import { openPonia } from '@ponia/widget';

// Call when user clicks deposit
openPonia({
  targetChain: 'base',
  targetToken: 'USDC'
});

Basic Configuration

Configure Ponia to match your application's needs using the options object:

openPonia({
  // Optional: 'deposit' (default) or 'withdraw'
  mode: 'deposit',
  
  // Required: Target blockchain
  targetChain: 'ethereum',
  
  // Required: Target token
  targetToken: 'USDC',
  
  // Optional: Pre-fill amount
  amount: '100',
  
  // Optional: User wallet address
  recipientAddress: '0x...',
  
  // Optional: Transaction callbacks
  onSuccess: (tx) => console.log('Success!', tx),
  onError: (error) => console.error(error)
});

API Reference

openPonia(options)

Opens the Ponia widget with the specified configuration.

Parameter Type Required Description
mode string No Transaction type: deposit (default) or withdraw
targetChain string Yes Destination blockchain. Supported: ethereum, polygon, arbitrum, base, optimism, bnb, solana, tron
targetToken string Yes Destination token. Options: NATIVE, USDC, USDT
amount string No Pre-fill transaction amount (in token units)
recipientAddress string No Recipient wallet address on target chain
onSuccess function No Callback when transaction completes successfully
onError function No Callback when transaction fails
onClose function No Callback when widget is closed

Callbacks

onSuccess

Called when the cross-chain transaction completes successfully.

onSuccess: (transaction) => {
  console.log('Transaction hash:', transaction.hash);
  console.log('Amount:', transaction.amount);
  console.log('From chain:', transaction.sourceChain);
  console.log('To chain:', transaction.targetChain);
  
  // Update your database, show success message, etc.
}

onError

Called when the transaction fails or is rejected.

onError: (error) => {
  console.error('Error type:', error.type);
  console.error('Message:', error.message);
  
  // Show error message to user
}

Example: NFT Marketplace

Accept payments from any chain for NFT purchases:

    targetToken: 'ETH',
    amount: price,
    recipientAddress: MARKETPLACE_WALLET,

      // Transfer NFT to buyer
      transferNFT(nftId, tx.senderAddress);

    onError: (error) => {
      alert('Payment failed. Please try again.');
    }
  });
}

FAQ

Which blockchains are supported?

Ponia supports 8 major deBridge-compatible blockchains: Ethereum, Polygon, Arbitrum, Base, Optimism, BNB Chain, Solana, and TRON.

How long do transfers take?

Average transfer time is 43 seconds. Maximum transfer time is typically under 60 seconds thanks to deBridge's Zero-TVL architecture.

What are the fees?

Ponia charges a flat 1.5% fee on all cross-chain transactions. No hidden costs, no setup fees, no monthly charges.

Is it safe?

Yes. Ponia is non-custodial, meaning users maintain full control of their funds at all times. We're powered by deBridge, which has undergone 30+ security audits with zero exploits.

Do I need an API key?

No. Ponia is completely plug-and-play. Just add the script tag and start accepting cross-chain payments.

Can users pay with any token?

Users can pay from any supported chain with any supported token (Native, USDC, USDT). Ponia handles the cross-chain routing automatically.

How do withdrawals work?

Withdrawals work exactly like deposits, but in reverse. Set mode: 'withdraw' and users can send funds from your platform to any supported blockchain. Perfect for casinos, gaming platforms, and exchanges.
